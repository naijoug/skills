# Trending - AI 领域热门话题收集

## 功能说明

收集和整理每日 AI 领域的热门话题、新产品、重要论文和行业动态。

## 使用方式

### 收集今日热门话题

```bash
alma trending collect
```

### 查看历史记录

```bash
alma trending show <date>
```

## 数据存储

每日话题存储在 `~/.config/alma/memory/ai-trending/` 目录下，按日期组织文件。

## 支持的信息源

- HN（Hacker News）
- Reddit r/MachineLearning
- arXiv 论文
- GitHub Trending（AI/ML 相关）
- Twitter/X（AI 领域 KOL）
