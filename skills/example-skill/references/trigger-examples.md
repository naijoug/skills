# Trigger Examples

Use these examples to test whether the `example-skill` triggers correctly.

## Positive (Chinese)

- 帮我创建一个新 skill 的模板
- 我想写一个新的 AI 技能，怎么开始？
- 提供一个 skill 的标准结构

## Positive (English)

- Create a new skill template for me.
- Show me the standard structure for a skill.
- I want to add a new skill to my collection.

## Negative / Near Miss

- Create a new file called README.md. (Generic file creation, not skill-specific)
- How do I install fzf? (Tool installation, not skill creation)
- List all my skills. (Listing action, not creation)
