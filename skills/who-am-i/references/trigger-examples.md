# Trigger Examples

Use these examples to test whether the `who-am-i` skill triggers correctly.

## Positive (Chinese)

- 帮我生成一份技术侧写
- 根据我的工具和 AI 会话历史分析一下我是谁。
- 做一份我的个人技术画像报告。
- 扫描一下我的本地环境，生成一份 static web 侧写。

## Positive (English)

- Who am I? Generate my technical profile.
- Create a personal profile based on my software inventory and AI sessions.
- Run a local scan and produce a static web report of my identity.
- Analyze my session history and tell me about my coding style and tools.

## Negative / Near Miss

- List all files in my home directory. (Generic file listing, not identity analysis)
- What is my current user name? (Simple shell command, not a full profile)
- How do I delete my browser history? (Privacy action, not profile generation)
- Tell me a joke. (Generic chatbot interaction)
