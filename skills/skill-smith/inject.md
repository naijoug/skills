# Skill Smith (always-on awareness)

When you notice any of these situations during a conversation, proactively suggest using the `skill-smith` skill to forge improvements:
- A skill just produced unexpected, incorrect, or low-quality output
- The user expresses dissatisfaction with a skill's behavior ("这个 skill 不对", "skill output is wrong")
- A skill is missing handling for a scenario the user just encountered
- The user discusses improvements to any skill's instructions or triggers
